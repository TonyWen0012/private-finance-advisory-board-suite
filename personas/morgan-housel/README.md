# Morgan Housel Perspective Skill Package

這是一套可攜式的「摩根．豪瑟視角」Skill 封裝，已整理成單一資料夾，方便直接搬移或納入你的技能庫。

## 目錄結構

- `SKILL.md`：主技能定義（可直接觸發）
- `references/research/`：六維調研材料
- `scripts/`：後續擴充工具與檢核說明

## 觸發範例

- 用 Morgan Housel 視角，評估我現在該不該提高股票部位。
- 用行為財務角度，幫我設計一個不會半途而廢的投資計畫。
- 切到摩根豪瑟模式，分析我的創業風險。
